# My Blog — Minimal Template

A clean, typography-focused blog built with Next.js and Tailwind CSS.

## Get started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Add posts

Edit `lib/posts.ts` to add or modify posts. Each post has a `slug`, `title`, `date`, `excerpt`, and `content`.
